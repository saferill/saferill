## 🧑‍💻 Tentang Saya

Halo! Saya **M. Syafril Ramadhani**, biasa dipanggil **Syaf**. Saya adalah seorang mahasiswa **Akuntansi di Universitas Nahdlatul Ulama Yogyakarta** (Angkatan 2024) yang memiliki passion besar di dunia teknologi dan kreativitas digital.

Perjalanan saya unik dan saling melengkapi:
* 🎓 **Latar Belakang Multimedia:** Lulusan SMK Nurul Jadid Paiton (2021-2024), menguasai desain grafis dan video editing.
* 💼 **Pengalaman Admin:** Pernah magang di bagian Administrasi Universitas Muhammadiyah Jember dan bekerja sebagai Social Media Admin.
* 🚀 **Fokus Saat Ini:** Menyatukan ketelitian prinsip Akuntansi dengan efisiensi **Web Development (React, Next.js)** untuk menciptakan sistem informasi yang modern dan mudah digunakan.
